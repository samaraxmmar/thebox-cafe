# Icônes PWA — The Box

3 fichiers requis dans ce dossier :

- **icon-192.png** — 192×192 px (standard)
- **icon-512.png** — 512×512 px (haute résolution)
- **icon-512-maskable.png** — 512×512 px avec padding sécurité 80% (centre du logo dans un cercle imaginaire)

## Génération rapide en ligne (gratuit)

1. Va sur **https://realfavicongenerator.net/**
2. Upload ton logo "THE BOX" (carré, PNG ou SVG, au moins 512×512)
3. Configure les options Android/iOS comme tu veux
4. Télécharge le pack
5. Récupère les 3 fichiers ci-dessus dans le pack et place-les ici

## OU plus simple : https://www.pwabuilder.com/imageGenerator

Upload une image carrée, ça génère TOUT le set automatiquement.

## Placeholder temporaire

En attendant les vraies icônes, tu peux générer une icône placeholder en ligne sur
https://favicon.io/favicon-generator/ avec :
- Text : `TB`
- Background : `#16a34a` (rond)
